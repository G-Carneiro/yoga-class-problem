# yoga-class-problem

---
## Objetivo

- Cinco mulheres estão participando de uma aula de yoga. Descubra qual é a posição favorita de yoga de cada uma delas.
- A solução deve ser encontrada utilizando prolog.

---
## Execução

- `swipl`
- `[yogaClass].`
- `solucao().`
